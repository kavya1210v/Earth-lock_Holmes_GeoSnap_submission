# GeoSnap Land-Use Classification & Explainability from Space
Team : Earth-lock Holmes

## Project Overview:
This project investigates land-use and land-cover classification using the EuroSAT benchmark dataset and deep learning techniques. Two transfer learning models based on ResNet18 were developed: one trained on RGB satellite imagery and another adapted to utilize all 13 Sentinel-2 multispectral bands. The objective is to evaluate how additional spectral information beyond the visible range influences classification performance and improves the discrimination of visually similar land-cover classes.

Both models were trained and evaluated using standard classification metrics, including accuracy, precision, recall, F1-score, confusion matrices, and classification reports. Beyond predictive performance, the project emphasizes model interpretability through explainability techniques such as Grad-CAM, feature-space visualization (t-SNE), spectral signature analysis, and band ablation studies. These analyses provide insight into the spatial regions and spectral bands that contribute most to the model's predictions.

The experimental results demonstrate that the multispectral ResNet18 model outperforms the RGB model, highlighting the importance of near-infrared and shortwave infrared information for remote sensing applications. Overall, this project illustrates the effectiveness of transfer learning and explainable AI techniques for accurate and trustworthy satellite image classification.

## Dataset
This project uses the **EuroSAT** dataset, a benchmark dataset derived from Sentinel-2 satellite imagery for land-use and land-cover classification.

Two versions of the dataset are used:

- **RGB Dataset** вЂ“ Uses the three visible spectral bands (Red, Green, Blue).
- **Multispectral Dataset** вЂ“ Uses all 13 Sentinel-2 spectral bands.

The official train-validation split is provided through `train.csv` and `validation.csv`, while separate flattened test datasets are used for evaluating both the RGB and multispectral models.

Classes:
- AnnualCrop
- Forest
- HerbaceousVegetation
- Highway
- Industrial
- Pasture
- PermanentCrop
- Residential
- River
- SeaLake

Input Types:
- RGB images (64Г—64)
- 13-band Sentinel-2 multispectral imagery
